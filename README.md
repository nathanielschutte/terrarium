# terrarium
Graphical python projects

## Use
- General

```
python run.py --program <program> [ --width 800 ] [ --height 600 ] [ --options <key=val> ... ]
```

- Conway's Game of Life example:

```
python run.py --program conway [ --options size=100 step=0.5 ... ]
```

## Help
`python run.py --help`
